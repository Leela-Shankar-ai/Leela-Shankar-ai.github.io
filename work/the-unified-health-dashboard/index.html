The Unified Health Dashboard

The situation

His health data lived in three apps that refuse to talk to each other: a WHOOP band, an OMRON smart scale and a meal tracker. So he built himself an Iron Man dashboard: every vital, on one screen, watched every single day.

What we delivered

  • WHOOP recovery, strain and sleep data connected over standard authentication
  • The OMRON scale had no public API at all, so its cloud API was reverse engineered
  • Meal data pulled from a tracker running on legacy authentication with manual request signing
  • All three pipelines run on our own automation infrastructure and feed one custom dashboard
  • An AI layer cross references every source and issues plain language verdicts, from recovery to nutrition

Why it worked

Reverse engineering an undocumented API for fun is exactly the muscle that client integrations depend on.
