How this site was made.

This whole experience was designed and built by Claude Fable, Anthropic's model, working autonomously: the concept, the shaders, the 3D, the content pipeline, the copy and the deployment. Here is the recipe, so you can do the same.

1. Content, not lorem

Everything true on this site was harvested straight from the company's real sales deck. A Python script read the deck generator's data structures, pulled 58 client engagements, and wrote them to a single content.json. The site is generated from that file, so the deck and the website can never drift apart.

2. An asset pipeline, not a download folder

Client logos, product screenshots and animated dashboards were converted for the web with Pillow and ffmpeg: logos to WebP at two tones (colour on white, and a knocked out version for dark plates), screenshots downscaled to WebP, and every scrolling GIF re encoded to a tiny looping MP4. The whole media library came down to about 13 MB.

3. A shader for atmosphere

The living background is a single fullscreen fragment shader in Three.js: domain warped fractal noise ramped through an ember palette, with a light pool that follows the cursor and recedes as you scroll. In light mode the same field inverts into warm paper. No video, no images, a few kilobytes of GLSL.

4. The Invariant, in 3D

The home hero is an icosahedron displaced by 3D simplex noise every frame: endlessly transformed, yet held together by a constant ember core and a calm wireframe shell. It is the company name made literal, orbited by drifting shards, reacting to the mouse.

5. Motion as a system

Reveals, split word headlines, magnetic buttons, tilt on every card, an ember trailing cursor, animated counters and an infinite logo marquee are all one small vanilla JS file driven by IntersectionObserver and requestAnimationFrame. No animation library. It respects prefers reduced motion.

6. Dual theme by design

Two full palettes live as CSS custom properties, so the toggle is a single attribute flip. Both the shader and the 3D scene read the theme and morph between them over about a second, rather than hard cutting.

7. Typework

Clash Display for expressive headlines, Zodiak italic as a contrast serif for the poetic lines, General Sans for body, JetBrains Mono for labels. The mix is deliberately editorial, not the default startup grotesk.

8. Generated and shipped

A second Python script renders every route to static HTML with shared nav and footer, then the whole thing deploys to Vercel as a static site. You are looking at the output.

Built with Three.js, vanilla JavaScript, hand written GLSL, Pillow and ffmpeg for assets, and a pair of Python generators. Fonts from Fontshare and Google Fonts. Deployed on Vercel. Zero front end frameworks.
